The RATCPU (Redstone Automated Turing Complete Processing Unit) is an 8 bit computer that I designed for fun. 
It was originally planned for a Minecraft CPU, hence the Redstone part, until I figured out that my computer is not at all strong enough to handle that big of a build.
Now, it just runs through a simulation I wrote in java for my APCSA final.
